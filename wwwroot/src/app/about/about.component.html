This is an MVC ASP.NET Core application which uses Angular 2 as a View Model. Both implementations use services to call an API and return a response as an interface.

People

The People search calls the PeopleController which uses the ExamplePeopleList to propagate the PeopleRepository.

Users

The Users search uses the RandomUser.me API to grab 5000 example users to search upon.
