About

This is an MVC ASP.NET Core application using Angular 2 for the View. Both implementations (People & User) use services to call an API and return a response as an interface.

People

The People search calls the PeopleController which uses the ExamplePeopleList to propagate the PeopleRepository.

Users

The Users search uses the RandomUser.me API to grab 5000 example users to search upon.

Features

  • Two views for "Users": Table and Card. Table view for efficiency and Card if an individual needed to distiguish a user by their picture.
  • Cell & Row highlighting for user focus & feedback.
  • Immediate search in "People" due to small data size.

'Tools' Used

  • Microsoft's Typescript
  • Jasmine for Testing
  • SASS for styling
